# CCS-Calculator
A much more functional CCS (Carbon Capture and Storage) Calculator

## Overview
This is a web application for calculating Carbon Capture and Storage metrics, deployed as an Azure Static Web App with Azure Functions backend.

## Getting Started

### Prerequisites
- Node.js (v18 or later)
- Azure Functions Core Tools
- Azure account (for deployment)

### Local Development
1. Clone the repository
2. Review and update `master-plan.md` with specific requirements
3. Install dependencies (once package.json is created)
4. Run the development server

### Documentation
- See `master-plan.md` for project planning and feature roadmap
- See `documentation/` folder for detailed as-built documentation
- See `.github/copilot-instructions.md` for development guidelines
