# Frontend Source

This folder contains the frontend source code for the CCS Calculator.

## Structure
- HTML files for the web interface
- CSS for styling
- JavaScript for client-side logic

## Development
Build your static web application here. The compiled/built output should be ready for Azure Static Web Apps deployment.

## Guidelines
- Keep code modular and maintainable
- Use modern JavaScript (ES6+)
- Ensure responsive design
- Test across browsers
